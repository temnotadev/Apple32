
# Apple32

ESP32 device that support "Apple Juice" attacks at Apple and Android devices


## Screenshots

![Device](https://i.ibb.co/bjVqtx6d/image.png)

## Information
This is an ESP32 hack project, this device has: AppleJuice (Apple, Android, Win11) and TV-B-Gone.
Device was tested on Apple: IOS26, IOS18, and IOS17 (On IOS17 phone crashes)
Android: 11, 12, 13, 14, 15, 16 (Everything works but on never versions of android its slower)
TV-B-Gone: Sony-TV

## Features

- Apple Juice attacks
- Attack mode selector
- IOS 26 support
- Android Spam
- Windows 11 Spam
